# Multivers - Applications Perso 🚀

Ce dépôt contient une suite d'applications modulaires gérées par un launcher central élégant.

## 📦 Contenu
- **Launcher Universel** : Interface CustomTkinter pour accéder à tous vos outils.
- **CERviable** : Automatisation de rapports CER (Prosit -> CER).
- **Jarvis Native** : Assistant IA (Ollama/Gemini) avec recherche web.
- **Downloader Universel** : Téléchargement de médias (YouTube, TikTok, etc.).

## 🚀 Installation
Pour installer la suite, téléchargez la dernière version du **Web-Installer** dans les [Releases](https://github.com/EstebanZGL/Multi-Apps/releases).

L'installateur vous permettra de choisir quelles applications installer et s'occupera de la configuration automatique.

## 🛠️ Pour les développeurs
1. Clonez le dépôt.
2. Installez les dépendances : `pip install -r requirements.txt`.
3. Pour ajouter une application, consultez le guide dans `GEMINI.md`.
4. Utilisez `python build.py` pour un build rapide ou complet.
5. Utilisez `python pack_installer.py` suivi de `python deploy_builds.py` pour publier de nouvelles versions d'apps.
